# Project Title: DMoney REST API Automation Testing
## Project Description: This project contains automated API testing for the DMoney REST API using Postman, Newman, and JavaScript test scripts.
It includes test cases, a Postman collection, Newman reports, API documentation, and bug reports as required for the assignment.
#### About Dmoney:
DMoney is a fake money transaction portal designed for learning and testing financial operations.
It simulates a real mobile financial service (MFS) system where users can perform essential financial transactions.


### Project Overview:

- This repository covers the end-to-end testing workflow of the DMoney financial API, including:
- User creation (Agent, Customer, Merchant)
- Money transactions (Deposit, Send money, Withdraw, Payment)
- Balance check and transaction statements
- Positive & Negative test cases
- Newman HTML report generation
- API documentation
- Bug/improvement report

  
### Technologies Used:

- Postman
- Newman
- Node.js
- JavaScript (Test scripts)
- HTML Report (htmlextra)
- Google Sheets
  
### How to run:
- `nmp i`
- `npm test`
